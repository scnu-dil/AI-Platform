# 源码
>down下后，按以下命令执行

```
//进入down后的AI-Platform/backend目录
~$ virtualenv --no-site-packages venv
//安装前期已有的依赖包
~$ pip install -r ../requirement.txt
~$ source venv/bin/activate
//直接运行manager.py
（venv）python manager.py

//开启另一终端，启动前端
具体操作移步至frontend阅读README
```
